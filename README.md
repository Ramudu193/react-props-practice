# React Redux Practice Project

This project is a simple React application created to practice **Redux state management**.  
It demonstrates how to create a store, manage global state, dispatch actions, and access data using Redux in a React application.

---

## 🚀 Features

- React functional components
- Redux store creation
- Actions and reducers
- Global state management
- Updating state using dispatch
- User information display using Redux
- Clean folder structure

---

## 🛠️ Technologies Used

- React JS
- Redux
- JavaScript (ES6)
- Vite
- CSS
